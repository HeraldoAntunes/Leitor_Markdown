\# Leitor Markdown



Aplicativo local em HTML para abrir, visualizar e testar textos em Markdown diretamente no navegador.



\## Objetivo



Este projeto foi criado para facilitar a leitura e a visualização de arquivos Markdown, especialmente materiais didáticos, anotações de aula, instruções de atividades e documentos simples.



\## Como usar



1\. Baixe ou clone este repositório.

2\. Abra o arquivo `index.html` no navegador.

3\. Insira ou carregue o texto em Markdown.

4\. Visualize o resultado renderizado.



\## Características



\- Funciona localmente no navegador.

\- Não exige servidor.

\- Usa um único arquivo `index.html`.

\- Pode ser usado para revisar materiais didáticos em Markdown.
